Matlab software for

A Perceptual Quality Metric with Internal Generative Mechanism

J. Wu, W. Lin, G. Shi, A. Liu
Dec 30, 2011
==========================================

This toolbox implements the algorithm described in: 

J. Wu, W. Lin, G. Shi, A. Liu, Perceptual Quality Metric with Internal
Generative Mechanism, IEEE Trans. on Image Processing, 2012.

------------------------------------------------------------------
The code package includes

1) Matlab Codes

2) Two images from TID2008 database

3) The evaluation results on 6 publicy avaliable databases
